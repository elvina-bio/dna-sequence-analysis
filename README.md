# DNA Sequence Analysis

A Python tool for analyzing DNA sequences. It counts nucleotides (A, T, G, C) and calculates GC-content.

## Features

- Counts A, T, G, C nucleotides
- Calculates GC-content percentage
- Simple and clean output

## Example Input

ATGCGATACGCTTGA

## Example Output

A: 4  
T: 4  
G: 4  
C: 3  
GC-content: 46.67%
